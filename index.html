SteelCalc — open-source steel weight & quotation calculator. Weights use cross-section × length × density; beam sections use the plate-area method (ignores root fillets, so real rolled sections run ~2–4% heavier — check the mill table for critical work).
round bar = π/4·d²·ρ  ·  pipe = π/4·(OD²−ID²)·ρ  ·  RHS/SHS = [HB−(H−2t)(B−2t)]·ρ  ·  plate = L·W·t·ρ
