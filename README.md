# SteelCalc

🚀 An open-source steel weight and quotation calculator for engineers, contractors, fabricators, and procurement professionals.

## Features

- MS Pipe Weight Calculator
- RHS / SHS Weight Calculator
- H-Beam Calculator
- I-Beam Calculator
- MS Plate Calculator
- Round Bar Calculator
- Cost Estimator
- Unit Converter
